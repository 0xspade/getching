# getching
Enhanced get-title script of tomnomnom
* With Status Code
* Blank Title
* No Title

# Install / Update

`go get -u github.com/phspade/getching`

# Usage

`echo "http://example.com" | getching`

or

`echo "http://example.com" | getching -follow-redir`
It should be combine with httprobe
